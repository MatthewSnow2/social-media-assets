New Source Modules

Files added to src/ and dashboard/ beyond the original ClaudeClaw codebase
R

router.ts

Multi-LLM message router. Parses @prefix syntax and dispatches to the right backend. Default routes to Claude Code.

Live
L

llm-backends.ts

HTTP client implementations for Gemini, Perplexity, Ollama (local), and OpenAI. Shared JSON request helper.

Live
C

classifier.ts

Three-tier message classification: quick patterns, long indicators, worker routing (Starscream, Ravage, Soundwave).

Built
P

result-poller.ts

10-second interval poller for completed async tasks. Sends results back to the user via Telegram with worker labels.

Built
A

arcade-config.ts

Arcade MCP gateway config. Bridges HTTP to stdio for Linear, GitHub, and Slack tool access inside Claude Code sessions.

Live
T

telemetry.ts

JSONL event stream: messages received, routes taken, agent completions, tool usage. Consumed by Sky-Lynx and ST Factory.

Live
D

dashboard/

Python report generator + HTML dashboard. Reads git status, service health, pipeline, Soundwave data, and Metroplex queue.

Live
S

Soundwave Section

Dashboard module reading 4 ST Factory sources: improvement recs, Metroplex priority queue, persona patches, research signals.

Live
